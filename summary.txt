1. Buat direktori => mkdir
2. Berpindah direktori => cd
3. Inisialisasi projek node dengan opsi default => npm init -y
4. buka vscode di direktori => code direktori
5. lengkapi package json
6. pilih license, bisa di lhat di internet (ex. choosealicense.com)
7. Buat file LICENSE, isikan dengan informasi license yang didapat di langkah 6
8. Buat file entrypoint berdasarkan protperty main di dalam packag.json
9. buat script start yang berisikan node namaFileKalian
10. Running script => npm start
11. tambahkan readme.md
